Import Images

In this content, I am going to import an image from the server using image element. This element requires the location of the image for which I have used a url of the image. You can link any image from your own PC by using the location. Make sure to properly define the location. If the image is on desktop, you need to describe the proper location to the image or my personal suggestion is to send that image to same folder as of your html file. By this, You need not to decribe the whole location and only name is sufficient to call the image.

It's important to resize the image. We can do this in CSS also. But for now, I am going to illustrate this in a simple way for the fact that we can do thsi on HTML also.

One more important thing. Makke sure to use alt attribute while importing the image. Although, this won't create any change in the method of importhing the image. But this will enable the feature in which, our differently abled friends, who are using the screen reading software and are unable to see the images can understand what's the content of that image. So, If you are using a Screen reader, you can hear "Ocean Tide waves". Also, I have used strong element in this paragraph to make this content noticable for the screen readers.
